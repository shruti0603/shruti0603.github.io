# Shruti Halde Portfolio

Simple static portfolio ready for GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `resume.pdf`

## Deploy to GitHub Pages
1. Create a GitHub repository or open your existing portfolio repo.
2. Upload all files from this folder to the root of the repository.
3. Commit and push.
4. In GitHub repo settings, open **Pages**.
5. Set source to the main branch root folder.
6. Save and wait for deployment.

## Customize
- Edit content in `index.html`
- Adjust colors and spacing in `styles.css`
- Add interactions in `script.js`
